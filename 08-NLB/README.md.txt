# ⚖️ Network Load Balancing

## 🧠 Objective
To distribute client requests across multiple servers.

## ⚙️ Steps
- Install NLB feature.
- Create NLB cluster and add nodes.
- Configure port rules and affinity.

## 🧩 Verification
- Access the virtual IP from a browser.

## 📝 Notes
Keep consistent configurations on all nodes.
