# 🧩 Failover Clustering

## 🧠 Objective
To configure a Windows Server cluster for continuous availability.

## ⚙️ Steps
- Install Failover Clustering role.
- Validate cluster configuration.
- Create and configure cluster shared volumes.
- Test failover.

## 🧩 Verification
- Run `Get-ClusterNode` and `Get-ClusterGroup`.

## 📝 Notes
Use static IPs for cluster networks.
