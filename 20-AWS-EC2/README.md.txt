# 💻 AWS EC2 Management

## 🧠 Objective
To manage Windows EC2 instances effectively.

## ⚙️ Steps
- Launch, connect, and manage instances.
- Configure instance roles and policies.
- Monitor CPU and disk via CloudWatch.

## 🧩 Verification
- Check instance state and health checks.

## 📝 Notes
Tag instances for easy identification.
