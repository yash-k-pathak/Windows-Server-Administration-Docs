# ⚙️ Managing Hyper-V Virtual Machines

## 🧠 Objective
To install and manage Hyper-V virtual machines on Windows Server.

## ⚙️ Steps
- Install Hyper-V role.
- Create virtual switches (External/Internal/Private).
- Create new VMs and attach ISO images.
- Configure checkpoints and snapshots.

## 🧩 Verification
- Run `Get-VM` in PowerShell to verify.

## 📝 Notes
Use Generation 2 VMs for newer OS compatibility.
