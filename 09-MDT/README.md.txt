# 🚀 Microsoft Deployment Toolkit (MDT)

## 🧠 Objective
To automate OS deployment using MDT.

## ⚙️ Steps
- Install MDT and Windows ADK.
- Create a deployment share.
- Import OS, drivers, and applications.
- Configure Task Sequence.
- Boot client with LiteTouch ISO.

## 🧩 Verification
- Successful automated OS deployment.

## 📝 Notes
MDT simplifies mass deployments in enterprises.
