# 🧠 Group Policy Management

## 🧠 Objective
To centrally manage settings using GPOs in Active Directory.

## ⚙️ Steps
- Open Group Policy Management Console.
- Create and link GPOs.
- Configure user and computer policies.
- Test using `gpupdate /force`.

## 🧩 Verification
- Check applied GPOs using `gpresult /r`.

## 📝 Notes
Organize policies by OU for clarity.
